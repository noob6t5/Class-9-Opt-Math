# Class-9-Opt-Math
Animated/offline-ready math visualizers for topics of class-9-opt math



## 🚀 How to View the Interactive Lessons?

You can view these modules directly in your browser without downloading anything by using a "GitHub HTML Previewer."

### 🌐 Option 1: Direct Link (Recommended)
Click the link below to open the **Mapping** lesson directly:
* [Open Mapping Lesson](https://htmlpreview.github.io/?https://github.com/noob6t5/Class-9-Opt-Math/blob/main/Algebra/4_mapping.html)

### 🔗 Option 2: Manual Link Construction
If you want to open any other file from this repository, use this format:
`https://htmlpreview.github.io/?` + `[Link to  GitHub HTML file]`

**Example:**
- Visit :`https://htmlpreview.github.io/`  
- Copy the url of repo `https://github.com/noob6t5/Class-9-Opt-Math/blob/main/Algebra/5_vetical_line.html` like this
- Paste that URl in the placeholder .

### 📂 Option 3: Offline View (Local)
If you have cloned or downloaded this repository to your computer:
1. Open your File Manager.
2. Navigate to the `Algebra` or `Matrix` folder.
3. **Double-click** the `.html` file (it will open in your default browser like Firefox or Chrome).
